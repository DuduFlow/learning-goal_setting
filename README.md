# 目標設定學習教案網頁

這是簡宏鈞處經理「目標設定」課程的第二版學習筆記網頁。

## 檔案結構

```
goal_setting_learning_project/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── goal-hero.png
```

## 本版重點

- 逐字稿與講義內容交叉整理成教案型筆記；講師投影片僅作內容對照，不包含於成品。
- 純 HTML/CSS/JS，無外部 CDN 依賴。
- 包含拖延診斷、SMART 策略、21 天飛輪承諾、五人圈盤點、文化勾選與下半年行動稿。
- 表單內容會保存於瀏覽器 localStorage。
- 可匯出文字作業檔，適合課後回填與追蹤。
- 頂部導覽含「返回目錄」按鈕，可回到總目錄頁：https://duduflow.github.io/learning_notes_catalog/

## 模板規格

後續每堂課筆記頁都建議保留頂部「返回目錄」按鈕，讓學員可在單堂筆記與總目錄之間來回切換。

## 使用方式

直接開啟 `index.html` 即可瀏覽；整個資料夾可直接上傳 GitHub Pages。
